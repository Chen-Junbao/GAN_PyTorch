# GAN_PyTorch
Various GAN implementations based on PyTorch.

## Environment
The code is developed using Python 3.6 on Windows 10. NVIDIA GPUs are needed. The code is developed and tested using NVIDIA GeForce GTX 1060.

## Requirement
```
torch == 1.1.0
torchvision == 0.3.0
CUDA == 9.0
cuDNN == 7.0
matplotlib == 3.0.2
```
